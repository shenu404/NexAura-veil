## рҹӣ пёҸ Manual Setup

### Requirements
- Ubuntu 20.04+ / Debian 11+
- Node.js 18+
- A domain pointing to your VPS

### Step 1 вҖ” Clone
```bash
git clone https://github.com/shenu404/nexaura-veil.git /opt/nexaura-veil
cd /opt/nexaura-veil
```

### Step 2 вҖ” Xray
```bash
bash -c "$(curl -L https://github.com/XTLS/Xray-install/raw/main/install-release.sh)" @ install
```

### Step 3 вҖ” Backend
```bash
cd backend
npm install
cp .env.example .env
nano .env   # Set JWT_SECRET + FRONTEND_URL
npm start
```

### Step 4 вҖ” Frontend
```bash
cd ../frontend
npm install
npm run build
```

### Step 5 вҖ” PM2
```bash
npm install -g pm2
cd ../backend
pm2 start src/index.js --name nexaura-veil
pm2 save && pm2 startup
```

### Step 6 вҖ” Nginx + SSL
```bash
# Copy nginx.conf вҶ’ /etc/nginx/sites-available/nexaura-veil
# Edit: replace YOUR_DOMAIN with your domain
sudo ln -s /etc/nginx/sites-available/nexaura-veil /etc/nginx/sites-enabled/
sudo nginx -t && sudo systemctl reload nginx
sudo certbot --nginx -d yourdomain.com
```

## рҹ”’ IP Limit System

Veil enforces a **strict single-IP policy** per client config:

```
Client connects       вҶ’ IP locked for that client
Same IP reconnects    вҶ’ вң… Allowed (same device)
Different IP connects вҶ’ рҹҡ« Auto-banned for 1 hour
IP silent 30+ min     вҶ’ вң… New IP allowed (network switch)
After 1 hour          вҶ’ вң… Auto-unbanned
Admin manual ban/unban вҶ’ Available from dashboard
```
